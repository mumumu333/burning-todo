## Burning Todo

## 概要
このアプリはTODOリストを管理するアプリです。タスクを追加して、「完了」ボタンを押すと、タスクがアニメーションで燃えながら削除されます。  
![image](https://github.com/user-attachments/assets/087dd185-9dcf-4b93-8fde-2553649a3547)

## 使用方法
- タスクの詳細と期限を入力して、「追加」ボタンをクリックします。  
![image](https://github.com/user-attachments/assets/c5b10c54-14a1-494a-a758-38e50d1ad8eb)

- todoリストにタスクが追加されます  
![image](https://github.com/user-attachments/assets/9fdf48ac-024a-4335-93bb-3054c15464e1)

- 完了ボタンを押すと、タスクが燃えて消えます。  
![image](https://github.com/user-attachments/assets/29ed1467-6970-42d7-9de6-02929df8cb4b)  
![image](https://github.com/user-attachments/assets/81f4b703-ed03-4431-b4a0-8c2d6020a841)  
![image](https://github.com/user-attachments/assets/38604e11-9b40-425e-9a3d-9950c3b69cb6)  




## 必要な環境
- Node.js
- npm または yarn

## セットアップ方法
1. リポジトリをクローンします：
   `git clone https://github.com/mumumu333/burning-todo.git`

2. プロジェクトフォルダに移動します：
   `cd repository-name`

3. 必要なパッケージをインストールします：
   `npm install` または `yarn install`

4. アプリを起動します：
   `npm start` または `yarn start`

5. ブラウザで `http://localhost:3000` にアクセスして、アプリが動いているか確認します。
