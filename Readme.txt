===========================
Burning Todo アプリ
===========================

## 概要
このアプリはTODOリストを管理するアプリです。タスクを追加して、「完了」ボタンを押すと、タスクがアニメーションで燃えながら削除されます。

## 必要な環境
- Node.js
- npm または yarn

## セットアップ方法
1. リポジトリをクローンします：
   `git clone https://github.com/mumumu333/burning-todo.git`

2. プロジェクトフォルダに移動します：
   `cd repository-name`

3. 必要なパッケージをインストールします：
   `npm install` または `yarn install`

4. アプリを起動します：
   `npm start` または `yarn start`

5. ブラウザで `http://localhost:3000` にアクセスして、アプリが動いているか確認します。

## 使用方法
- タスクの詳細と期限を入力して、「追加」ボタンをクリックします。
- タスクが追加され、完了ボタンを押すと、タスクが燃えて消えます。

## 使用している技術
